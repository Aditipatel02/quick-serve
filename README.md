# quickserve

Quickly build inference engines from jupyter notebooks and deploy on web and edge in 3 lines of code

## Get started

* Clone the repo!

```sh
$ git clone https://github.com/pranjaldatta/quick-serve.git
```

* Install with pip

```sh
$ pip install .
```
